# John Hurr Portfolio

A simple static portfolio site built for GitHub Pages.

## Upload to GitHub

1. Open your `USERNAME.github.io` repository.
2. Upload everything inside this folder to the repository root.
3. Commit the changes.
4. The site should publish automatically.

## Before publishing broadly

- Replace the two placeholder images in `/images` with approved, high-resolution assets.
- Replace the placeholder LinkedIn URL in `index.html`.
- Replace `#` project links with real case-study pages.
- Confirm that `/resume/John-Hurr-Resume.pdf` is the résumé you want public.
- The site currently loads Inter from Google Fonts. It can be changed to a self-hosted font later.

## Structure

- `index.html`
- `styles.css`
- `/images`
